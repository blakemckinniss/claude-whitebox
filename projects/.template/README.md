# Project Template

## Quick Start
1. Copy this template: `cp -r projects/.template projects/my-project`
2. Update this README with project details
3. Begin development

## Directory Structure
- `src/` - Source code
- `tests/` - Test files
- `docs/` - Documentation
- `data/` - Data files (gitignored by default)

## Development
All claude-whitebox tools are available from project root.
Run commands from `/home/jinx/workspace/claude-whitebox/`
